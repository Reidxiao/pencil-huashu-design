## 🚨 硬性禁止（一旦发现，必须改）

### 1. 色彩禁止

| ID | 禁止模式 | AI slop 典型表现 | 为什么会 slop |
|----|---------|----------------|--------------|
| C01 | 紫渐变背景 | `fill: linear-gradient(#8B5CF6, #A855F7)` | AI 最喜欢的"科技感"配色 |
| C02 | 彩虹渐变 | 多色渐变，蓝→紫→粉 | 过度花哨，没有品味 |
| C03 | 霓虹蓝/绿 accent | `#00D4FF`、`#39FF14` | 赛博朋克刻板印象 |
| C04 | 高饱和蓝色主调 | `#3B82F6` 大面积使用 | AI 生成的默认"信任蓝" |
| C05 | 纯黑/纯白背景 | `#000000` 或 `#FFFFFF` 纯色 | 没有温度感，太冷 |
| C06 | 灰色渐变背景 | 灰到更灰的渐变 | 懒得配色时的默认选项 |
| C07 | 青色/蓝绿色调 | `#06B6D4`、`#14B8A6` | AI 偏好的"小众"科技色 |
| C08 | 粉紫少女色 | `#EC4899`、`#F472B6` | 不分场景的"女性化"用色 |

### 2. 排版禁止

| ID | 禁止模式 | AI slop 典型表现 | 为什么会 slop |
|----|---------|----------------|--------------|
| T01 | Inter 做 display | 大标题用 Inter | AI 最熟悉的字体 |
| T02 | Roboto / Open Sans | 任何场景都用 | 没有品牌感 |
| T03 | 字体大小过于均匀 | 标题 32，正文 30，注释 28 | 不讲层级 |
| T04 | 行高过紧 | line-height: 1.2 | 读起来很累 |
| T05 | 字间距过宽 | letter-spacing: 4px 标题 | 刻意拉大显得空洞 |
| T06 | 所有文字居中 | 标题居中、正文居中、按钮居中 | 懒 |
| T07 | 全大写标题滥用 | HERO / FEATURES / ABOUT | 不分场景的"专业感" |
| T08 | 过多字体混用 | > 3 种字体混排 | 没有视觉秩序 |

### 3. 布局禁止

| ID | 禁止模式 | AI slop 典型表现 | 为什么会 slop |
|----|---------|----------------|--------------|
| L01 | 圆角 + 左边框 accent | 卡片 `border-radius: 12px` + 左边 4px accent 色 | 最常见的 AI 卡片设计 |
| L02 | 等分布局强迫症 | 所有卡片 3 等分、间距完全相同 | 太规整，没有节奏感 |
| L03 | 大量留白的浪费 | 上下 200px padding，内容只有 20% | 用空白掩盖内容不足 |
| L04 | 居中癌 | 所有东西都水平垂直居中 | 懒 |
| L05 | Hero 区域过大 | Hero 占整个页面 80% | 内容贫乏的信号 |
| L06 | 卡片网格过于规整 | 2×2、3×3 完美网格 | 设计感为零 |
| L07 | CTA 按钮过大 | CTA 占屏幕 30% 宽度 | 粗暴，没有层次 |

### 4. 图形禁止

| ID | 禁止模式 | AI slop 典型表现 | 为什么会 slop |
|----|---------|----------------|--------------|
| G01 | emoji 当图标 | 😂 🔥 ✅ ❌ 💡 | 没有设计系统 |
| G02 | SVG 手画人脸 | 画得很丑的火柴人脸 | AI 不擅长画人 |
| G03 | 3D 按钮 + 阴影 | `box-shadow: 0 4px 12px` | 过时的拟物风格 |
| G04 | 饼图/折线图过于花哨 | 渐变填充、多色系 | 用图表装饰而非解释数据 |
| G05 | "科技感"网格背景 | 细线条透视网格 | 陈词滥调 |
| G06 | 抽象几何图案滥用 | 随意摆放的三角形/圆/线条 | 没有美感 |
| G07 | stock photo 滥用 | Unsplash 模特摆拍 | 假、不真诚 |
| G08 | AI 生成图像 | Midjourney 风格的"完美"图片 | 没有真实感 |

### 5. 交互禁止

| ID | 禁止模式 | AI slop 典型表现 | 为什么会 slop |
|----|---------|----------------|--------------|
| I01 | hover 效果过于夸张 | hover 整个元素变色+放大+阴影 | 廉价感 |
| I02 | 加载动画过于复杂 | 多层动画叠加 | 等待焦虑 |
| I03 | 滚动视差滥用 | 每一屏都有视差 | 眩晕 |
| I04 | 幽灵按钮（ghost button）滥用 | 透明背景+边框按钮到处用 | 没有主次 |

## ✅ 推荐做法（正面清单）

### 色彩替代方案

| 替代 | 用在哪 | 示例 |
|------|--------|------|
| 单一有品味的颜色 | 主色调 | `#1A1A1A`（近黑）、`#C4A484`（暖棕） |
| 暖灰/冷灰背景 | 背景 | `#F8F7F4`（暖白）、`#F0F4F8`（冷蓝灰） |
| 品牌色做点缀 | accent | 5% 面积使用，不大面积铺开 |
| 真实图片做主视觉 | Hero 背景 | Wikimedia、Met Museum、Unsplash |
| 细微纹理 | 背景 | 噪点纹理 opacity 0.03-0.08 |
| 中性色系 | 专业场景 | `#374151`（石板灰）、`#6B7280`（灰色） |

### 排版替代方案

| 替代 | 用在哪 | 示例 |
|------|--------|------|
| 衬线 display 字体 | 标题、H1-H3 | Newsreader、Source Serif、Noto Serif、Literata |
| 无衬线 body 字体 | 正文 | Inter（仅 body）、Söhne、Helvetica Neue |
| 字重对比 | 层级 | 标题 700，正文 400，注释 300 |
| 行高舒适 | 正文 | line-height: 1.5-1.7 |
| 适当字间距 | 大写标题 | letter-spacing: 0.05em |
| 左对齐正文 | 正文 | 除标题外，全左对齐 |

### 布局替代方案

| 替代 | 用在哪 | 示例 |
|------|--------|------|
| 非对称布局 | 内容区 | 左重右轻，或反之 |
| 网格 + 破格 | 打破规律 | 7 列网格中某个元素跨 3 列 |
| 不同尺寸的内容块 | 卡片组 | 有的 1x1，有的 2x1，有的 1x2 |
| 负空间（留白） | 分隔内容 | 不用线，用空白 |
| 水平节奏 | 列表/导航 | 基于 8px 网格 |
| 垂直节奏 | 段落间距 | 基于 24px / 32px / 48px |

### 图形替代方案

| 替代 | 用在哪 | 示例 |
|------|--------|------|
| Lucide / Material Symbols | 图标 | 线性风格、统一 stroke |
| 真实图片 | 产品/人物 | 不手画，不用 emoji |
| 手工感插图 | 装饰 | 不规则线条、纸张纹理 |
| 几何图形组合 | 背景装饰 | 圆形、三角形、线条的克制组合 |
| 品牌色应用 | 视觉识别 | 重复 logo 图形、颜色呼应是更高级的做法 |

## 🔍 检查流程

### Step 1：远看（5 秒测试）

- 第一眼看到的是什么？
- 整体氛围是"科技感 AI"还是"有品味的设计"？
- 有没有刺眼的紫/蓝/绿？
- 是不是 3 列等分网格？

### Step 2：近看（细节扫描）

逐项检查硬性禁止清单（见上方 C01-T08 / L01-L07 / G01-G08 / I01-I04）。

### Step 3：交互检查

- 层级是否清晰（第一眼→第二眼→第三眼）？
- 间距是否基于 8px 网格？
- 对齐是否精确？
- 暗色模式下颜色是否适配？

### Step 4：对照正面清单

确认使用了推荐做法，而非仅仅避免了禁止项。

## 📋 AI Slop 自检 Prompt

每次完成设计后，用这个 Prompt 自检：

```
请检查当前 .pen 文件中的设计是否有以下 AI slop 特征：

色彩类：
1. 紫渐变或彩虹渐变背景
2. 霓虹蓝/绿作为主色调
3. 高饱和蓝色大面积铺开
4. 纯黑/纯白全屏背景

字体类：
5. Inter/Roboto/Open Sans 用于 display 字体
6. 字体大小无层级对比

布局类：
7. 圆角卡片 + 左边框 accent 组合
8. 3 列/4 列完全等分网格
9. 所有元素居中

图形类：
10. emoji 当图标
11. 手绘 SVG 人脸
12. 科技感透视网格

交互类：
13. hover 效果过于夸张

如果有，请用 batch_design 修正这些元素。
```

## 案例对比

### ❌ AI Slop 示例

```
Hero 区域：
- 深色背景 + 紫色渐变
- "AI-Powered" 标语
- Inter Bold 标题
- 3 列等分网格，每个卡片完全相同尺寸

卡片：
- 白色圆角卡片 + 左边 4px #8B5CF6 accent
- 内容：emoji 🔥 + 标题 + 描述

字体：
- Inter Bold 标题
- Inter Regular 正文
- 标题 32，正文 30

图标：
- 🔥 💡 ✅ ❌ emoji 组合
```

### ✅ 有品味的设计

```
Hero 区域：
- 暖白背景 + 真实产品图
- 简洁衬线标题
- 左对齐，非对称布局

卡片：
- 部分浅灰背景，部分近白
- 无左边框 accent
- 颜色变化来自品牌色板

字体：
- Newsreader Display 标题
- Inter Body 正文
- 标题 48，正文 16，注释 14

图标：
- Lucide 图标库
- stroke-width 统一
```
