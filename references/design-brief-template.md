- **Why** (Problem solved / Goal): ________________
- **Where** (Device / Context): ________________
- **When** (Timeline / Urgency): ________________
- **How** (Delivery format / Preferences): ________________

#### Functional Requirements
1. ________________
2. ________________
3. ________________

#### Brand Constraints (if any)
- Logo: ✅ Have / ❌ None / ⚠️ Need to find
- Brand Colors: ________________
- Competitor References: ________________

#### Emotional Tone
- 😐 Cool & Professional / 😊 Warm & Friendly / 😎 Cool Tech / 🎨 Artistic / ❓ Uncertain

#### Hard Constraints
1. Must support dark mode
2. Cannot use certain colors/fonts: ________________
3. Must be accessible (WCAG AA)

#### My Assumptions (need user confirmation)
1. ________________
2. ________________

#### Pencil Capability Check
- [ ] Needs complex 3D/physics simulation → ⚠️ Pencil doesn't support
- [ ] Needs real-time dynamic data → ⚠️ Pencil produces static designs
- [ ] Needs drag interaction → ⚠️ Pencil prototyping is limited
- [ ] None of the above → ✅ Pencil is suitable

**Confirmed by user before proceeding to design.**

#### Pencil 能力边界判断
